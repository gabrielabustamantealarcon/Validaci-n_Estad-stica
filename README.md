# Validacion_Estadistica
Este proyecto forma parte de la estrategia de análisis de datos del equipo de marketing digital. El objetivo principal es evaluar estadísticamente un experimento A/B ejecutado en la página de inicio (landing page) para decidir, mediante evidencia sólida, qué versión (A o B) debe implementarse para maximizar la conversión y el valor económico por usuario.

🎯 Objetivo del Proyecto

Explorar, validar y analizar estadísticamente el experimento A/B para identificar diferencias significativas entre las páginas y responder a las siguientes preguntas clave del negocio:
* ¿Existe una diferencia significativa en el gasto promedio por usuario convertido?
* ¿Qué versión de la página (A o B) genera mayor tasa de conversión?
* ¿Influyen la fuente de tráfico o el tipo de usuario (nuevo o recurrente) en la conversión?
* ¿Qué hallazgos permiten optimizar la estrategia de marketing y el diseño de la landing page?

📊 Datasets Utilizados

El análisis se basa en el procesamiento de la siguiente fuente de datos:
* `landing_experiment.csv`: Información de usuarios expuestos a las versiones A y B, incluyendo variables de región, dispositivo, fuente de tráfico, tipo de usuario, conversión y gasto.

| Paso | Acción Técnica | Resultado para el negocio |
